# Security Audit Reports

Repository for access review evidence, SOX compliance artifacts, and audit-ready reports
generated via **Saviynt EIC** certification campaigns.

> ⚠️ All data in this repository is synthetic/anonymized. No real user data is stored here.

## Report Types

| Type | Frequency | Generated By |
|---|---|---|
| Manager Access Certification | Quarterly | Saviynt Campaign Export |
| Privileged Access Review | Monthly | PAM Audit Module |
| SOD Violation Summary | Quarterly | Saviynt AAG |
| Orphaned Account Report | Monthly | Automated Discovery |
| Terminated User Access Report | Weekly | Leaver Workflow Audit |

## Retention Policy
All audit artifacts retained for **7 years** per SOX and internal policy requirements.
